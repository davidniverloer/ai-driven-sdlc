# First Skill Proposal

This guide explains how to read the first skill proposal example.

Use it after installing ai-driven-sdlc and reading:

```text
.ai-sdlc/guides/FIRST_RUN.md
.ai-sdlc/commands/manifest.md
```

## Purpose

The first skill proposal example shows the minimum useful output shape for
`/skill-create`.

It demonstrates:

- source prompt preservation
- capability classification
- repository placement
- Process impact
- similarity analysis
- approval checklist
- validation report

## Installed Location

After installation, the example is available at:

```text
.ai-sdlc/examples/first-skill-proposal/
```

## How To Use It

Ask the agent to compare a generated proposal against:

```text
.ai-sdlc/examples/first-skill-proposal/
```

The generated proposal does not need to copy the example wording. It should
match the structure and preserve its own submitted source prompts verbatim.

## Boundary

The example is not a canonical skill and must not be promoted. It is a reference
package for structure, validation expectations, and safe proposal review.
