# Approval Checklist

## Prompt Fidelity Verification

- [x] Source prompt is preserved verbatim.
- [x] Source prompt order is preserved.
- [x] Source prompt structure is preserved.
- [x] Source prompt wording is preserved.

## Capability Verification

- [x] Capability classification is present.
- [x] Capability classification rationale is present.

## Repository Placement Verification

- [x] Proposed repository location is present.
- [x] Placement rationale is present.
- [x] Affected assets are listed.

## Process Verification

- [x] Process impact report is complete.
- [x] New or affected nodes are identified.
- [x] Artifact changes are identified.
- [x] Approval gate changes are identified.

## Governance Verification

- [x] Proposal lifecycle state is valid for an example.
- [x] Prompt Immutability requirements are satisfied.
- [x] Proposal Governance requirements are satisfied.

## Validation Verification

- [x] Skill Proposal Validation completed for example scope.
- [x] Compatibility impact assessed.
- [x] Breaking-change impact assessed.
- [x] Promotion readiness assessed.
