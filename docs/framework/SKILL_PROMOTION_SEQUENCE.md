# Skill Promotion Sequence

This document is the authoritative promotion flow for validated Skill Proposals.

## Canonical Sequence

```text
Proposal
↓
Validation
↓
Promotion Evaluation
↓
Approval
↓
Canonical Registration
```

## Stage Definitions

**Proposal**

A Skill Proposal package exists and contains required source, metadata,
classification, repository placement, Process impact, and approval
checklist artifacts.

**Validation**

The proposal has completed Skill Intake Validation and has a validation report.

**Promotion Evaluation**

The Skill Promotion Engine evaluates promotion criteria, compatibility impact,
governance compliance, and registration readiness.

**Approval**

Required governance approval is recorded. Protected operations still require
separate approval.

**Canonical Registration**

The promoted skill is admitted to the canonical skill area and registered once
the Canonical Skill Registry exists.

## Non-Goals

This sequence does not implement promotion logic, registry updates, skill
creation, workflow creation, schema validation, or git operations.
