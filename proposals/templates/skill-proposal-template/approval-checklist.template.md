# Approval Checklist

## Prompt Fidelity Verification

- [ ] Source prompts are preserved verbatim.
- [ ] Source prompt order is preserved.
- [ ] Source prompt structure is preserved.
- [ ] Source prompt wording is preserved.

## Capability Verification

- [ ] Capability classification is present.
- [ ] Capability classification rationale is present.

## Repository Placement Verification

- [ ] Proposed repository location is present.
- [ ] Placement rationale is present.
- [ ] Affected assets are listed.

## Process Tree Verification

- [ ] Process Tree impact report is complete.
- [ ] New or affected nodes are identified.
- [ ] Artifact changes are identified.
- [ ] Approval gate changes are identified.

## Governance Verification

- [ ] Proposal lifecycle state is valid.
- [ ] Proposal ownership is recorded.
- [ ] Prompt Immutability requirements are satisfied.
- [ ] Proposal Governance requirements are satisfied.

## Validation Verification

- [ ] Skill Proposal Validation completed.
- [ ] Compatibility impact assessed.
- [ ] Breaking-change impact assessed.
- [ ] Promotion readiness assessed.
