# Process Model

The canonical ai-driven-sdlc process is the Markdown execution map that defines
execution order, artifact flow, repository placement, approval gates, and
handoffs across the framework.

The active default canonical process lives at:

```text
processes/default.md
```

The legacy path `process-trees/default.md` is retained only as a compatibility
pointer.

Future agents must use:

```text
processes/default.md
```

as the canonical default process.

For process governance, use:

```text
docs/execution/PROCESS_POLICY.md
```

For process integration rules, use:

```text
docs/framework/PROCESS_INTEGRATION_RULES.md
```
