# Source Prompts

This directory contains immutable copies of submitted source prompts and
protocols.

Do not rewrite, optimize, improve, reinterpret, merge, paraphrase, expand,
compress, or reformat source prompt originals.

Recommended filenames:

```text
prompt.original.md
discovery.original.md
synthesis.original.md
step-001.original.md
step-002.original.md
```

Derived annotations, classifications, mappings, and proposal drafts must live
outside this directory.
